# web-starter

Website development kit w/ Gulp + coffee + sass + jade.

## Structure

```
.
├── README.md
├── bower.json
├── gulp
│   ├── config.coffee
│   └── tasks
│       ├── build.coffee
│       ├── clean.coffee
│       ├── copy.coffee
│       ├── markups.coffee
│       ├── publish.coffee
│       ├── scripts.coffee
│       ├── styles.coffee
│       └── watch.coffee
├── gulpfile.coffee
├── package.json
└── src
    ├── markups
    │   └── index.jade
    ├── scripts
    │   ├── MyModule.coffee
    │   └── index.coffee
    └── styles
        └── index.sass
```
